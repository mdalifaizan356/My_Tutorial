NPM understanding:
    Node js core me jo installed ho k aata he wo module kehlata he.
    Jo hum npm se download karte he wo package kehlata he.

    installing and unistalling from npm
        for install: npm i/ install packagename
        for install package version: npm i/ install packagename@version

        for unistall: npm unistall packagename

    understanding node_module folder
        Is folder me packages and packages ki dependencies hoti he.
    
        devdependencies: wo sab jo development k time tk useful he isme store hoti he. like nodemon.

    script: default script or custom script contain kar k rakhti he. default script nmp scriptName se run ho jati he.
            but custom script ko run karne k liye npm run scriptName use karte he.
